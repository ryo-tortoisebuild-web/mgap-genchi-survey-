# 現地調査 引き継ぎアプリ（MVP）

リフォーム・原状回復の現地調査情報を、現地に行かない職人へ正確に引き継ぐためのアプリ。
サーバー・外部API不要。ブラウザだけで完結します（HTML/CSS/JSのみ）。

**中心コンセプト**：間取りを地図にして、場所ごとに写真を格納し、職人ごとにその写真へ書き込みを載せ、職人ごとに1枚完結のPDFを出す。

## 3タブの流れ

1. **間取りタブ（土台）**
   「＋間取りを追加」→「🖼 背景」で間取り画像（募集図面のスクショ・写真・手書き撮影など）を読み込み、拡大・縮小・全体に合わせる・ドラッグで位置合わせ → 「＋撮影ポイントを追加」で名称・位置メモ・職人タグ・施工指示・写真（複数可）を登録 → 一覧の「配置」で間取り上の撮った場所に①②③…とピンを打つ。間取り上にも下記の書き込みツール一式が使えます。
2. **依頼先タブ（写真への書き込み）**
   「＋職人を追加」で依頼先（14職種）を選ぶ → その職人が担当する撮影ポイントの写真が並ぶ → 写真をタップして書き込みエディタで注釈を描く。**書き込みは職人ごとのレイヤー**として保存され、写真自体は1枚のまま。職人を切り替えると、その職人の書き込みだけが表示されます（クロス用・解体用…を同じ写真に重ねられる）。
3. **出力タブ（職人別PDF）**
   職人を選ぶ → 表紙（物件名・職人名）→ ①の写真【主役サイズ】＋その職人の書き込み＋位置メモ・施工指示 → ②… と①から並んだ依頼書 → 「PDF出力」→ 印刷ダイアログで「PDFとして保存」（PC Chrome推奨）。

### 書き込みツール（間取り・写真で共通）

- 図形（線／丸／四角）は「図形▾」ボタンから選択、ほかに 矢印／手書き／コメント（文字）／寸法＋数値
- 色（6色）と太さ（細・中・太）を選択可能
- **調整は「☝ 選択」で図形を直接つまむ方式**（ハンドルはタッチしやすい大きさ）
  - **四角＝四隅を個別ドラッグして台形化**（写真の奥行き・パースに沿わせる）
  - 丸＝角ハンドルでサイズ、辺ハンドルでつぶし（楕円化）、回転ハンドルで回転
  - コメント＝幅ハンドルで折り返し幅、高さハンドルで文字サイズ（自動折返し＋エンターで手動改行、行が増えると枠が下に伸びる）
  - 図形本体のドラッグで移動
- コメント入力はエンターで登録されません（改行／IME変換確定用）。登録は「保存」または枠の外をタップ。既存コメントは再タップで文字を編集
- 寸法は撮影ポイントのフォームには持たず、写真への「寸法」書き込みに一本化

### データの持ち方

- 写真は撮影ポイントに1回だけ格納（職人ごとに複製しない）
- 書き込みは `写真ID × 職人タグ` のレイヤー（`photo.annotations[職人タグ]`）として保持
- 依頼先（職人）リストは物件ごとに保持。JSONは「1ファイル＝1物件」で後方互換（旧形式も読み込み可）

### 現地→事務所の受け渡し

現地でスマホ入力 → ヘッダー「⬇ JSON書き出し」→ AirDrop・LINE・ドライブ等でPCへ → PC側で「⬆ JSON読み込み」（物件として一覧に追加）。

## 複数物件の管理

- ヘッダーの「📁 物件一覧」から物件の作成・切替・削除ができます
- 物件を選ぶと、その物件の全データ（要素・図面・写真）がまるごと開き、修正・再PDF出力ができます
- 切替時、編集中のデータは自動保存されます
- JSONは従来どおり「1ファイル＝1物件」。読み込むと一覧に追加され、他の物件は消えません（同じ物件のJSONなら更新扱い）

## 起動方法

- **ローカルで試す**：`index.html` をブラウザで直接開くだけで動きます（file://対応）
- **スマホで使う**：フォルダごとWebサーバー（GitHub Pages等の静的ホスティング）に置き、スマホでURLを開く。ホーム画面に追加するとアプリのように使えます
- 開発用ローカルサーバー例：`npx serve -l 8642 .`

## データについて

- 変更は自動保存されます（ブラウザ内のIndexedDB）
- **正式な保存はJSON書き出し**です。現場作業後は必ず書き出してください（ブラウザのデータは端末やブラウザ削除で消えることがあります）
- 写真は登録時に自動縮小（長辺1280px）されJSONに埋め込まれます。1ファイルで受け渡し完結

## 構成

```
index.html          … 本体（間取り・依頼先・出力の3タブ＋情報カード）
css/style.css       … 画面UI（レスポンシブ）
css/print.css       … 職人別依頼書PDFレイアウト（A4縦・写真主役）
js/constants.js     … カテゴリ4系統・職人タグ14種・状態・書き込み色
js/store.js         … データ管理・自動保存・JSON入出力・旧データ自動移行
js/ui.js            … タブ・モーダル・トースト・入力ダイアログ
js/photo.js         … 写真の縮小・圧縮・実寸取得
js/draw.js          … 共通書き込みコア（描画・ツールバー・変形。写真/間取り共用）
js/input.js         … 撮影ポイントのフォーム（位置メモ・職人タグ・施工指示・写真）・物件管理・JSON入出力
js/editor.js        … 間取りタブ（SVG：背景・ピン・書き込み）
js/card.js          … 情報カード
js/annotate.js      … 依頼先タブ＋写真への書き込みエディタ（職人別レイヤー）
js/output.js        … 職人別PDF（写真主役＋書き込み）
manifest.webmanifest / sw.js / icons/ … PWA用
```
